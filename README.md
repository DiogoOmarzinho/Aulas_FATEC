# 🚀 Repositório de Aulas — DSM

### Desenvolvimento de Software Multiplataforma

Bem-vindo(a) ao nosso repositório oficial das aulas de **Desenvolvimento de Software Multiplataforma (DSM)**! 🎓💻

Este espaço foi criado para centralizar **códigos, exercícios, projetos, anotações e materiais complementares** desenvolvidos ao longo do curso.

---

## 📚 Sobre o Curso

O curso de **DSM** tem como objetivo formar profissionais capazes de desenvolver aplicações para múltiplas plataformas, utilizando boas práticas de engenharia de software, metodologias ágeis e tecnologias modernas.

Durante as aulas, trabalhamos com:

* 🧠 Lógica de Programação
* 🏗️ Arquitetura de Software
* 🌐 Desenvolvimento Web
* 📱 Desenvolvimento Mobile
* 🗄️ Banco de Dados
* 🔗 APIs e Integrações
* 🧪 Testes e Boas Práticas
* 🚀 Deploy e Versionamento

## 🤝 Como Contribuir

1. Faça um fork do repositório
2. Crie uma branch com sua feature:

   ```bash
   git checkout -b minha-feature
   ```
3. Commit suas alterações:

   ```bash
   git commit -m "feat: descrição da alteração"
   ```
4. Envie para o repositório remoto:

   ```bash
   git push origin minha-feature
   ```
5. Abra um Pull Request 🚀
